Freddy Jump!
This is a game where you control Freddy using WASD to avoid ghost and possibly reach the red door.
Initially, you will be given 5 chances and 90 seconds to finish the game. Every time you hit a ghost, you lose one life.
Two ways to lose the game: 1.life drops to 0   2.Times up
Three levels in total.

Files needed to run the game:
-Hack112Project.py
-cmu_112_graphics.py
-BG.png
-smallGhost.png
